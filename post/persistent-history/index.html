Persistent History 30 May 2016

Eli Bendersky has a neat script for keeping a persistent record of all commands run in bash. I was covetous and wanted the same thing but for zsh which turned out to be surprisingly easy:

# Insert into your .zshrc
log_persistent_history() {
    echo $(history -i -1) >> $HOME/.persistent_history
}

touch $HOME/.persistent_history
add-zsh-hook precmd log_persistent_history

This is a lot simpler than Eli’s script mostly because it doesn’t de-duplicate the commands (I’m actually kind of interested to see if I ever run ls twenty times in a row).
